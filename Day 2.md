# Registry and ADS
## REGEDIT
- open regedit gui
- click file
- connect to network.
## REG KEYS
- HKLM: Local Machine
- HKCR: Current Root - everyime system boots load it into rom
- HKCU: Current User - Symbolic Link with SID of Current User
## REG Commands
- get-item
- new-item
- set-itemProperty
### SID
